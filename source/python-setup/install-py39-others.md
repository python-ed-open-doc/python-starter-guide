## Homebrew(macOSのみ)
Homebrewはコマンドラインで開発用の便利なツールやライブラリをインストールできるパッケージ管理ツールです。
Homebrewのインストール方法は以下公式サイトを参照してください。

[https://brew.sh/index_ja](https://brew.sh/index_ja)

Python 3.9をインストールする場合は`brew install python@3.9`を実行します。
